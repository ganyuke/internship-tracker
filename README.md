# Internship Tracker

Web app to make recording applications to internship opportunities easier for me. Like a superpowered spreadsheet. Huge.

Super broken, super vibe-coded (thanks Gemini) internship tracker. But it gets the job done, so whatever.

Svelte and Sveltekit are so cool by the way.

## How to use

1. Install `pnpm` (or whatever NPM package manager you want).
2. Clone this directory.
3. Run `pnpm dev`.

I might have forgot a database initialization and migration step. Something to do with `drizzzle`.
